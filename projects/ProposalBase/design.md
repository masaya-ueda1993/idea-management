# ProposalBase 設計書

---

## Phase 1：要件定義 ✅

### 目的・背景

自治体への委託・指定管理者プロポーザルに参加する法人が、提案書作成を効率化するためのWebアプリ。  
自社の強み・実績をDB化し、案件ごとに再利用することで、毎回ゼロから書く手間をなくす。

現状の課題：
- 自社情報（強み・実績）が散在していて毎回探す手間がかかる
- 案件ごとに提案書をゼロから作っていて非効率
- 自治体ごとの案件情報・進捗がバラバラに管理されている
- 提案書の構成は自治体が公開する雛形に従う必要があり、毎回確認が必要

### ターゲットユーザー

自治体へのプロポーザルを繰り返し行う法人・企業の担当者

具体的には：
- NPO・社会福祉法人・民間企業など
- 年に複数回プロポーザルに参加する組織
- 提案書作成を担当する営業・企画職

### 解決する課題

| 課題 | 解決方法 |
|------|---------|
| 自社情報が散在している | 自社ナレッジDBに一元管理 |
| 毎回ゼロから提案書を書く | 過去文章を引用して再利用 |
| 案件の進捗管理が煩雑 | 案件管理機能で一元管理 |
| 締切を見落とす | ダッシュボードで締切を可視化 |

### v1 主要機能一覧

| # | 機能 | 概要 |
|---|------|------|
| 1 | 案件管理 | 自治体名・案件名・締切・ステータス・募集要項URL・雛形ファイル添付 |
| 2 | 提案書作成 | 自治体の雛形に沿ったセクション登録・本文入力・自社DBからの引用 |
| 3 | 自社ナレッジDB | 強み・実績・会社概要の登録・タグ検索 |
| 4 | 過去提案書閲覧 | 案件ごとの提案書履歴・検索 |
| 5 | ダッシュボード | 進行中案件一覧・締切カレンダー |

### 提案書作成の考え方

- 提案書のセクション構成は**自治体が公開する雛形で決まる**（ユーザーが自由に決めるものではない）
- ユーザーは雛形を参照しながらセクション名を手動登録する
- 各セクションに本文を入力し、**自社ナレッジDBから過去文章を引用**できる
- これがこのアプリの最大の差別化ポイント

### v2 以降（将来機能）

| バージョン | 機能 |
|-----------|------|
| v2 | 自治体ごとのテンプレート管理 |
| v2 | チームメンバーとの共有・権限管理 |
| v3 | AIによる文章提案（過去データ活用） |
| v3 | 自治体情報の自動取得 |

### 機能一覧（優先度付き）

| # | 機能名 | 詳細 | 優先度 |
|---|--------|------|--------|
| 1 | ダッシュボード | 進行中案件一覧・締切が近い順に表示 | 高 |
| 2 | 案件登録 | 自治体名・案件名・締切日・ステータス・募集要項URL・雛形ファイル添付 | 高 |
| 3 | 案件一覧 | 登録した案件の一覧・ステータスフィルター | 高 |
| 4 | 提案書作成 | 案件に紐づくセクション登録・本文入力（テキストのみ） | 高 |
| 5 | 自社ナレッジDB登録 | 強み・実績・会社概要をカテゴリ・タグ付きで登録 | 高 |
| 6 | 自社ナレッジDB一覧 | カテゴリ・タグ・キーワードで検索・引用ボタン | 高 |
| 7 | 過去提案書閲覧 | 案件ごとの提案書を閲覧・検索 | 中 |
| 8 | ユーザー認証 | ログイン・ログアウト・会員登録 | 高 |

### 提案書エディタの方針（v1）

- **テキスト管理に徹する（アプローチA）**
- セクション構成は自治体の雛形に従いユーザーが手動登録
- 各セクションにテキストエリアで本文入力
- 自社ナレッジDBからモーダルで引用・挿入できる
- 画像・表・リッチコンテンツはv2以降で対応

### ナレッジ引用の仕組み

```
提案書作成画面のセクション入力中
  → 「ナレッジから引用」ボタンをクリック
  → モーダルが開く（画面遷移なし）
  → カテゴリ・タグ・キーワードで絞り込み検索
  → 「引用する」ボタンでテキストエリアに挿入
  → そのまま編集して仕上げる
```

### 画面遷移

```
ログイン
  └─ ダッシュボード（トップ）
       ├─ 案件一覧
       │    ├─ 案件登録
       │    └─ 案件詳細
       │         ├─ 案件編集

       │              └─ ナレッジDB（モーダルで呼び出し）
       └─ 自社ナレッジDB一覧
            ├─ ナレッジ登録
            └─ ナレッジ編集
```
