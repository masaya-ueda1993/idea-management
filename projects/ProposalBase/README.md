# ProposalBase

自治体への委託・指定管理者プロポーザルに特化した、提案書作成・ナレッジ管理Webアプリ。

## 概要

自社の強み・実績をDB化し、案件ごとに再利用することで、毎回ゼロから提案書を書く手間をなくす。

## ターゲット

自治体へのプロポーザルを繰り返し行う法人・企業の担当者

## 技術スタック（予定）

- Backend: Laravel
- DB: MySQL
- Frontend: Blade（Laravel標準）
