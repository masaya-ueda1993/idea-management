# AI COMPASS 設計書

---

## Phase 1：要件定義

### 目的・背景

AIツールの選択肢が増え「自分にはどれが合うか・いくらかかるか」が分かりにくくなっている。
数値スコアでの比較はバイアスが入りやすく、料金と性能を別々に調べる手間もある。
→ **用途と頻度を入力するだけで、向いているAIと必要プランを一度に提示する**ツールを作る。

### ターゲットユーザー

- AIツールを使い始めたい一般ユーザー
- 複数のAIを使い分けているが整理したいユーザー
- 料金プランで迷っているユーザー

### 解決する課題

| 課題 | 解決方法 |
|---|---|
| どのAIが自分の用途に向いているか分からない | 用途×AI の強み・弱み（◎△✕）で比較 |
| 数値スコア比較はバイアスがある | スコア廃止・定性評価に統一 |
| 性能と料金を別々に調べる手間がある | 1フローで両方を同時に提示 |
| 用途によって使う頻度が違う | 用途ごとに頻度を個別設定できる |
| 最新情報が反映されていない | Claude API + web_search で毎回最新情報を取得 |

### 主要機能一覧

| # | 機能 | 説明 |
|---|---|---|
| 1 | 用途選択 | 6カテゴリから複数選択可能 |
| 2 | 頻度設定 | 用途ごとに5段階（たまに〜ヘビー）で設定 |
| 3 | AI比較 | Claude / ChatGPT / Gemini を◎△✕で評価 |
| 4 | 料金シミュ | 頻度から月間負荷を計算し必要プランを提示 |
| 5 | AI分析 | Claude API + web_search で最新情報を反映した詳細コメント |
| 6 | 結果表示 | 比較結果・料金を1画面に統合して表示 |

### 用途カテゴリ（固定）

| ID | ラベル | 説明 |
|---|---|---|
| coding | コーディング・開発 | コード生成・デバッグ・設計 |
| writing | 文章作成・ライティング | ブログ・メール・レポート |
| analysis | 調査・分析・要約 | リサーチ・データ整理・要約 |
| creative | アイデア・創作 | 企画・ブレスト・クリエイティブ |
| business | ビジネス・資料作成 | 提案書・プレゼン・議事録 |
| general | 日常・雑談・なんでも | 日常的な質問・調べ物 |

### 対応AI（固定）

| AI | 運営 | 対応プラン |
|---|---|---|
| Claude | Anthropic | Free / Pro / Team |
| ChatGPT | OpenAI | Free / Plus / Pro |
| Gemini | Google | Free / Advanced / Business |

### 非機能要件

| 項目 | 内容 |
|---|---|
| 公開対象 | 一般公開 |
| レスポンス | AI分析は10〜30秒を許容（web_search込み） |
| 対応デバイス | PC・スマホ両対応（レスポンシブ） |
| 共有・保存 | 対応しない（シンプルに保つ） |
| 言語 | 日本語のみ |
