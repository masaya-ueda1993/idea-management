# AI COMPASS

> 使い方と頻度から、最適なAIと料金を見つけるWebアプリ

## 概要

Claude / ChatGPT / Gemini を**数値スコアではなく強み・弱みで比較**し、
用途ごとの使用頻度から**必要プランと月額料金を一緒に提示**するツール。
